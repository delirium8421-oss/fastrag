# FastGraphRAG Pipeline Run Commands

## Quick Test (1% corpus, 1% questions) - ~30 seconds

```bash
python run_fast-graphrag.py \
  --subset medical \
  --mode ollama \
  --model_name qwen2.5-coder:3b \
  --embed_model_path nomic-embed-text:v1.5 \
  --llm_base_url http://127.0.0.1:11434 \
  --corpus_fraction 0.01 \
  --questions_fraction 0.01
```

## Troubleshooting

### If Ollama is not running:

```bash
# Start Ollama service
ollama serve
```

### If model is not available:

```bash
# Pull the model first
ollama pull qwen2.5-coder:3b
ollama pull nomic-embed-text:v1.5
```

### Check available models:

```bash
ollama list
```

### View logs in real-time:

```bash
tail -f graphrag_medical.log
```

## Output Location

Results are saved to:
```
./results/fast-graphrag/Medical/predictions_Medical.json
```

## Notes

- Gleaning is enabled (max_gleaning_steps=1) for better quality
- Logging shows INFO level (progress messages) and ERROR level (validation failures)
- Use fraction parameters for quick iteration during development
- Use full corpus (no fractions) for production evaluation
